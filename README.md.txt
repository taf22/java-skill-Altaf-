# Java Skills Project - Altaf

This is a simple Java project created to practice basic Java programming skills.

## 📁 Project Structure
- `src/` — contains the Java source files.
- `out/` — contains compiled `.class` files.
- `compile.bat` — a script to compile Java files.
- `README.md` — project description.
- `.gitignore` — excludes unnecessary files from Git.

## 🧪 What It Does
This project currently contains a basic Java program:
- `HelloWorld.java` prints a welcome message to the console.

## 🛠 How to Compile
Double-click the `compile.bat` file to compile all `.java` files in the `src` folder.

## 🚀 How to Run
After compiling, open Command Prompt and type:
java -cp out HelloWorld
